\# ShootingGame



A WPF shooting game built in C# for the .NET Technologies midterm exam at Sheridan College.



\## How to Play

\- \*\*← →\*\* Arrow keys to rotate the gun

\- \*\*Up Arrow\*\* or \*\*Space\*\* to shoot

\- Shoot shapes 5 times to destroy them

\- Avoid letting shapes touch the gun!



\## Features

\- 3 shape types: Circle, Rectangle, Triangle

\- Shapes bounce off all 4 walls

\- Shapes bounce off each other

\- Bullets destroy shapes with explosion effect

\- Audio and visual explosion effects

\- HP system — shapes change color as they take damage

\- Gun can be destroyed by shapes (bonus)



\## Tech Stack

\- C# / WPF (.NET 10)

\- DispatcherTimer for game loop

\- SoundPlayer for audio

\- Canvas for rendering
